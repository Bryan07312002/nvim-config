## Bryan neovim configuration
This is my neovim configuration, heavily inspired by ThePrimegen configs.

### Prerequisite
- [ripgrep](https://github.com/BurntSushi/ripgrep).
- lua 5.1
- luarocks

### How to install
Just clone this repository in you nvim config path, normaly at `~/.config/nvim`

example:
``` sh
git clone https://github.com/Bryan07312002/nvim-config.git ~/.config/nvim
```
